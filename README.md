# Indicina Spool SDK

- [Typescript](https://www.typescriptlang.org/)

## Installation

``````bash
npm i indicina-spool-sdk
``````

## Usage Example

``````bash

PopupSDK.openPopup(
    'https://www.example.com/widget',
    'spool@gmail.com',
);
  
setTimeout(() => {
    PopupSDK.closePopup();
}, 5000);
``````
